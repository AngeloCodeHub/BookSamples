機器學習 (基礎數學篇)

在徹底研究機器學習後，筆者體會許多基礎數學不是不會與艱難而是生疏了，如果機器學習的書籍可以將複雜公式從基礎開始一步一步推導，其實可以很容易帶領讀者進入這個領域，同時感受數學不再如此艱澀，這也是我撰寫本書時時提醒自己要留意的事項。

Python最強入門 (王者歸來)
