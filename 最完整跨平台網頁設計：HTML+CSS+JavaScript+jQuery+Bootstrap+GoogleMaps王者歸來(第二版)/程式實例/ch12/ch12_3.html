2006年2月為了享受邊泡溫泉邊看極光(Northern Light)， 一人獨自坐飛機至阿拉斯加(Alaska)，再開車往北至接近北極圈的 Chena Hot Springs溫泉渡假村。旅遊期間嘗試開車直達北極海(Arctic Ocean) ，第一次車子在冰天雪地打滑，撞山壁失敗而返。
